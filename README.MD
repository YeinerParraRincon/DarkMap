# 🔥 DarkMap - Port Scanner

**DarkMap** es una herramienta de escaneo de puertos desarrollada en Python, enfocada en aprendizaje de ciberseguridad y hacking ético.

---

## 🚀 Características

* Escaneo de puertos TCP
* Multithreading (rápido)
* Soporte para dominios (ej: google.com)
* Interfaz en terminal con banner personalizado
* Uso mediante argumentos (CLI)

---

## 🛠 Tecnologías

* Python 3
* socket
* argparse
* ThreadPoolExecutor

---

## ⚙️ Instalación

```bash
git clone https://github.com/tuusuario/darkmap.git
cd darkmap
python3 darkmap.py -i 127.0.0.1
```

---

## 🧪 Uso

```bash
python darkmap.py -i 192.168.1.1
```

o

```bash
python darkmap.py --ip google.com
```

---

## 📌 Ejemplo de salida

```
[+] Puerto 22 ABIERTO
[+] Puerto 80 ABIERTO
```
